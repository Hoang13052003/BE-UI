
## 📋 **ENDPOINT ĐÁNH DẤU FEEDBACK ĐÃ ĐỌC**

### **🔗 URL & Method:**
```http
PATCH /api/feedbacks/{feedbackId}/read
```

### **🔐 Authentication:**
- **Required**: Bearer Token trong header
- **Authorization**: `Bearer <your-jwt-token>`

---

## ✅ **CÁC RESPONSE CODES:**

### **Success (200 OK):**
```json
{
  "message": "Feedback marked as read successfully"
}
```

### **Unauthorized (401):**
```json
{
  "status": "UNAUTHORIZED",
  "message": "User not authenticated"
}
```

### **Forbidden (403):**
```json
{
  "status": "FORBIDDEN", 
  "message": "Access Denied: You don't have permission to mark this feedback as read"
}
```

### **Not Found (404):**
```json
{
  "status": "NOT_FOUND",
  "message": "Feedback not found with ID: 60a7b8c9d1234567890abcde"
}
```

---

## 🔒 **PHÂN QUYỀN:**
Các phân quyền này đã được cấu hình bên backen
| **Role** | **Có thể đánh dấu feedback nào?** |
|----------|-----------------------------------|
| **ADMIN** | ✅ Tất cả feedback |
| **MANAGER** | ✅ Feedback trong project họ quản lý |
| **USER** | ✅ Feedback của họ<br>✅ Feedback trong project họ tham gia |
